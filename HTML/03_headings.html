Headings

by Eric Washburn

Heading tags are a great tool for attracting focus and prioritizing content.

headings are also used as an accessibility tool for visually impaired users.
Therefore it is important to keep headings relevant to the content they preface.

There a six heading sizes and are typed out as follows:

    <h1>
    The only level 1 heading on this page is "WASH CODE" in the body header.
    <h2>

    this is a level 2 heading

    <h3>

    this is a level 3 heading

    <h4>

    this is a level 4 heading

    <h5>
    this is a level 5 heading
    <h6>
    this is a level 6 heading
