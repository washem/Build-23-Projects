Tables can be created with the <table> tag
Tables are constructed with rows (<tr>) and headers (<th>) and descriptions (<td>)
table headers and descriptions are nested within table rows.

here is the code example:

<table>
<tr>
<th>header</th>
<th>header</th>
<th>header</th>
</tr>
<tr>
<td>description</td>
<td>description</td>
<td>description</td>
</tr>
<tr>
<td>description</td>
<td>description</td>
<td>description</td>
</tr>
</table>

see the example below for the resulting output:

name age city
Mohamad 30 Baghdad
Fatima 21 Erbil
