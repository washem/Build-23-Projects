Lists

by Eric Washburn

The most common types of lists are "unordered", "ordered" and "descriptive" lists.

unordered and ordered lists will be indented, however description lists will not.

In the description list example below I've added <blockquote> to match the indentation.

Unordered List example:

  • carrots
  • patates
  • kale

Ordered list example:

  1. apples
  2. oranges
  3. bananas

descriptive list example:

kale
description of kale
orange
description of orange
carrots
description of carrots
