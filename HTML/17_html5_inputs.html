HTML5 Inputs

Association

The Label and input below are now associated thanks to the label's "for" attribute, which can be linked to the input's "id" attribute

in this case they are linked by "fname"

<label for="fname">first name</label>

<input type="text" id="fname">

HTML5 also adds the ability to select the input field by clicking on the label!

Validate

some inputs can be validated for proper formating like email and url.

simply add the "require" attribute to the input.

Input Types

Search

The new html "search" type: < input type="search" >

allows you to clear the search field after something has been typed into it!

Email

The new html "email" type: < input type="email" >

validation using the attribute "require" would check to see if a valid email format was entered

URL

The new html "url" type: < input type="url" >

validation using the attribute "require" would check to see if a valid url format was entered

Number

The new html "number" type: < input type="number" >

"min" and "max" attributes allow for a range of numbers to be selected.

the "step" attribute can also be used to increment the numbers by a set amount

omiting attributes will simply let the user select which ever integer they chose.

< input type="number" id="form-number" min="0" max="10" step="2">

Range

The new html "Range" type: < input type="range" >

will not display the number chosen. needs javascript.

"min" and "max" attributes allow for a range of numbers to be selected.

Date

The new html "Date" type: < input type="date" >

input field will display a calandar drop down (not all browsers)

date "min" and "max" can be enabled
