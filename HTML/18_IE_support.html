Legacy Browser compatibility

You can teach older browsers to handle HTML5 correctly.

Thankfully, Sjoerd Visscher created the HTML5Shiv!

HTML5Shiv is a JavaScript workaround to enable styling of HTML5 elements in versions of Internet Explorer prior to version 9.

the shiv/shim script

The following HTML5Shiv script must be placed in the <head> element, after any stylesheets:

<!--[if lt IE 9]>
<script src="https://oss.maxcdn.com/libs/html5shiv/3.7.0/html5shiv.js"></script>
<![endif]-->

this script tag is wrapped in a conditional comment that IE will read.

similar to a normal <-- --> comment, a conditional comment includes an "if" statement.

it reads out like this:

<!--[if lt IE 9]> = (if less that IE)
