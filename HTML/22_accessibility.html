Semantic HTML

Semantic HTML gives context to screen readers, which read the contents of a web page out loud.

using a <nav> instead of <div id="nav"> is example of practicing good semantic HTML.

Alternative text for images

The alt attribute provides an alternate text for an image, if the user for some reason cannot view it (because of slow connection, an error in the src attribute, or if the user uses a screen reader).

an example of an alt attribute that describes the image:

<img src="img_html_code.jpg" alt="html_code">
