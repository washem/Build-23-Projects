Semantic HTML

by Eric Washburn

The use of semantic markup can be very useful for search engine optimization (SEO). Semantic HTML is also good practice for understanding the meaning of a page and its layout.

an example of semantic html architecture:

  • <header>
  • <nav>
  • <main>
    • <article>
      • <header>
      • <section>
        • <aside>
      • <footer>
    • <aside>
  • <footer>
