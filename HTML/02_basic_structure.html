Index.html Basic Structure

by Eric Washburn

An HTML document is structured with building blocks refered to as "tags".

Web browsers read HTML documents and render them thanks to these tags.

The basic structure of any HTML document is as follows:

    <!DOCTYPE>
    <html>
    <head>
    <meta>
    <title>
    </head>
    <body>
    </body>
    </html>
