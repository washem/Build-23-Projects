Search Engine Optimization

Descriptive Title Tag

The <title> tag in the <head> section describes the page and is displayed in browser tabs. Ideally the page title and website name appears here.

Typically, search engines will look through page titles for relevant information. Good titles will generate better search results.

an example of content description in a <title> tag :

<title>SEO | WASH CODE - search engine optimization</title>

created on March 8th 2019 - By Eric Washburn

Meta Description Tags

The content attribute in a <meta> tag is used by search engines to display information about that website. Search engines will display a certain character limit, so it is important to curate this information in order to concisely summarize the website content. Matching this description to the <title> will greatly improve results

an example of content description in a <meta> tag :

<meta name="description" content="This is an example of a meta description. This will increase search engine optimization.">

created on March 8th 2019 - By Eric Washburn

Descriptive Header Tags

When <h1> tags (h1 to h6) are used in a topical manner, search engines will know to display them and their content as search results.

an example of content description in a <h1> tag :

<h1>Header Tags</h1>
