External CSS

by Eric Washburn

A seperate .css document will be created and referenced by this html document.

in order to access this CSS document, it will need to be linked in the <head> tag.

It should look like this:

<link rel="stylesheet" type="text/css" href="style.css">

this new CSS document can now contain style markups applied directly to each html tag:


    body {
        background-color: gray;
        color:#333
        }
    h1  {
        font-size: 50px;
        font-family: helvetica, Arial, sans-serif;
        }
