Inline CSS example

by Eric Washburn

Styles can be applied directly to an HTML document by using the "style" attribute for any html tag.

see the code example below:

<p style="color: orange; border: solid 2px blue; width: 350px"></p>

which will look like:

Paragraph tag containing orange text inside a blue box

***This method of styling html tags directly in the html document is no longer considered best practice.***
