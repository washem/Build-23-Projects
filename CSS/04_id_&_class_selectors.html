IDs and Class Selectors

by Eric Washburn

HTML tags can have specific markup instructions using the following attributes:

id=""

and

class=""

Here is an example:

<body id="idSample" class="class-sample"></body>

Simply preface the id="" attribute label in CSS with a #
in order to specify style instructions to that HTML tag: 


        #idSample {
            background-color: gray;
            color:#333
            }

Preface the class="" attribute label in CSS with a .(period)
in order to specify style instructions to that HTML tag: 


        .class-sample {
            background-color: gray;
            color:#333
            }

The HTML tag can be called before the ID or Class in order to be more specific
in the case that the ID or Class is used in different HTML tags: 


        code#idSample {
            background-color: gray;
            color:#333
            }
